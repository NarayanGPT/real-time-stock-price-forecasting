# real-time-stock-price-forecasting
A machine learning-based system for real-time stock price forecasting using historical market data, technical indicators, and predictive analytics.

# Dynamic Stock Price Forecasting using Machine Learning and Real-Time Data Integration

## Overview
This capstone project focuses on forecasting stock prices using Machine Learning and Deep Learning techniques.

## Models Used
- LSTM
- Random Forest
- Decision Tree
- Support Vector Regression (SVR)

## Dataset
- Historical stock data from Yahoo Finance
- TCS stock data (2010–2024)

## Results
| Model | Accuracy |
|---------|---------|
| LSTM | 96% |
| Random Forest | 92% |
| Decision Tree | 86% |
| SVR | 77% |
